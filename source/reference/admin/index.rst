アプリケーション全体の設定
==========================

「管理」画面では、プロジェクトやユーザーを作成したり、全般的な設定をしたりといったRedMica全体に関わる管理を行います。

RedMicaのシステム管理者権限を持っているユーザー（デフォルトではadmin）のみがトップメニューの「管理」からアクセスできます。
